Visualización de Gráficas Generadas a partir del Dataset

Introducción

Este proyecto visualiza los datos de un dataset sobre estudiantes, proporcionando sobre el rendimiento estudiantil en un programa de matemáticas. Proporciona información detallada tanto sobre el rendimiento académico. Las gráficas son generadas usando Python y almacenadas en la carpeta output.

Gráfica 4: Relación entre tiempo de estudio y calificación final

Gráfica 5: Correlación entre variables numéricas

Proceso de Carga y Graficación

El proceso de carga de datos se realiza mediante el script carga.py, que utiliza Elasticsearch para indexar los datos del archivo math-students.csv. Luego, las gráficas se generan mediante el script graficar.py, que utiliza bibliotecas de visualización como Matplotlib para crear las representaciones gráficas.
